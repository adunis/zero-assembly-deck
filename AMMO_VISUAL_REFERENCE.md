This guide describes what each ammunition type should look like for image creation.

**File**: `standard-bolt.png`  
**Description**: Single crossbow bolt projectile  
**Visual Elements**:

- Simple, clean design

## 🎨 DESIGN GUIDELINES

### Color Coding
- **Standard/Kinetic**: Orange, yellow, white, gray
- **Explosive**: Red, orange, black with warning stripes
- **Energy**: Blue, cyan, electric colors (for future ammo)
- **Hazards**: Red, orange, black with warning symbols

### Size Hierarchy
1. **Small**: Crossbow bolts, Mini Rockets, Hazard Markers
2. **Medium**: Standard Shell, Explosive Shell
3. **Large**: Standard BIG Shell, Explosive BIG Shell
4. **Extra Large**: Nuclear Bomb (unique)

### Visual Distinction
Each ammo type should be **instantly recognizable** at a glance:
- Different shapes (bolts vs spheres vs rockets)
- Different sizes (small vs medium vs large)
- Different colors (standard vs explosive)
- Different quantities (x1 vs x2 vs x3 vs x4)

### Style Consistency
- Clean, simple designs
- Clear silhouettes
- Easy to identify at small sizes (icon size)
- Professional appearance
- Consistent art style across all ammo types

## 📐 IMAGE SPECIFICATIONS

### Technical Requirements
- **Format**: PNG with transparency
- **Dimensions**: 200x200 pixels
- **DPI**: 72 (web standard)
- **Color Mode**: RGB
- **Background**: Transparent
- **File Size**: < 50KB per image

### Composition
- Center the projectile(s) in the frame
- Leave small margin around edges
- Use consistent lighting/shading
- Add subtle drop shadow for depth
- Include quantity indicators (x2, x3, x4) if applicable

---

## 🛠️ CREATION METHODS

### Option 1: Photography
- Take photos of actual 3D-printed projectiles
- Use white background
- Remove background in photo editor
- Add labels/indicators

### Option 2: 3D Rendering
- Render 3D models of projectiles
- Export with transparent background
- Add post-processing effects
- Add labels/indicators

### Option 3: Digital Illustration
- Draw/illustrate projectiles in vector or raster
- Use consistent art style
- Export at correct dimensions
- Add labels/indicators

### Option 4: AI Generation
- Use AI image generation (Midjourney, DALL-E, Stable Diffusion)
- Prompt: "game icon, [projectile description], transparent background, simple design"
- Post-process to add labels
- Ensure consistency across all images
