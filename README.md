# Zero Assembly Deck

A tactical tabletop wargame featuring customizable mechs in strategic combat.

## About

Zero Assembly Deck is a physics-based tabletop game where players build and command mechs using modular components. The game features:

- **Modular Mech Building**: Customize your mech with various weapons, equipment, and components
- **Tactical Combat**: Strategic gameplay with positioning, line of sight, and terrain interaction
- **Keyword System**: Streamlined rules using clear keywords for abilities and effects
- **Multiple Factions**: Choose from different factions, each with unique equipment and playstyles
- **3D Printing Support**: Print your own mech components and terrain

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/zero-assembly-deck.git
   cd zero-assembly-deck
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   node server.js
   ```

4. Open your browser and navigate to `http://localhost:3000`

## Project Structure

- `/public` - Web interface and game data
  - `/data` - JSON files containing equipment, weapons, and game content
  - `index.html` - Main game interface
- `server.js` - Express server for hosting the game
- Various `.js` scripts - Development tools for content management
- `.md` files - Documentation and development notes
