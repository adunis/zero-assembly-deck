## RECOMMENDATIONS

### For Best Results
1. **Print on good quality paper** - Smaller fonts need crisp printing
2. **Use high DPI printer** - 600 DPI minimum for 6pt fonts
3. **Test print first** - Print one page to verify readability
4. **Adjust if needed** - If too small, increase font sizes by 0.5-1pt

### If Still Too Small
Increase these values in `public/index.html`:
```css
/* Rules */
body { font-size: 9pt !important; } /* was 8pt */
.rule-text { font-size: 9pt !important; } /* was 8pt */

/* Cards */
.card { font-size: 7pt !important; } /* was 6.5pt */
.card-description { font-size: 6.5pt !important; } /* was 6pt */
```

### If Need Even More Compact
Reduce these values:
```css
/* Rules */
@page { margin: 6mm; } /* was 8mm */
.rule-section { margin-bottom: 3mm !important; } /* was 4mm */

/* Cards */
.deck-container { gap: 3mm !important; } /* was 4mm */
.card { padding: 1.5mm !important; } /* was 2mm */
```
