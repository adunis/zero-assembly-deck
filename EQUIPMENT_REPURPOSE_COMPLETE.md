# ✅ EQUIPMENT REPURPOSE COMPLETE

## Removed Duplicates & Repurposed Unique 3D Models

**Problem:** Multiple equipment cards using the same image, but each image represents a unique real-world 3D model.

**Solution:** Repurposed existing unique models with new mechanics and removed duplicate variants.

### 2. Repurposed Equipment with New Mechanics

#### A. Sentinel Sensor Array Head → Targeting Computer Head

**Old (eq_4):**
- Name: 'Sentinel' Sensor Array Head
- Effect: +1 to hit (once per turn)
- Image: Unique sensor array model

**New (eq_4):**
- Name: **'Targeting' Computer Head**
- Effect: **+1 to LOCK rolls** (Lock-On weapons only)
- Purpose: Makes establishing locks easier
- Image: Same unique sensor array model

**Why:** Sensor array fits better as a targeting computer that helps lock onto targets. The +1 to hit was redundant with Argus-Eye Head.

---

#### B. AMAP-L Light Plating → Reactive Armor Plating

**Old (eq_3):**
- Name: AMAP-L Light Plating
- Effect: +1 HP
- Image: Unique armor plating model

**New (eq_3):**
- Name: **'Reactive' Armor Plating**
- Effect: **Reduces 3 DP to 2 DP** (blocks 1 DP from heavy weapons)
- Uses: 2 per game
- Image: Same unique armor plating model

**Why:** Armor plating fits better as reactive armor that explodes outward to reduce damage. The +1 HP was redundant with Canopy Sealer.

---

#### C. Fortress Carapace → Fortress Heavy Armor

**Old (eq_5):**
- Name: Fortress Carapace
- Effect: +1 HP + connector
- Image: Unique heavy armor model

**New (eq_5):**
- Name: **'Fortress' Heavy Armor**
- Effect: **+2 HP** but **-5cm movement**
- Still a connector part
- Image: Same unique heavy armor model

**Why:** Heavy armor should provide more protection but slow you down. Creates interesting trade-off between defense and mobility.

---

## 🎯 NEW EQUIPMENT MECHANICS

### 1. Targeting Computer Head (eq_4)

**Type:** Utility System / Head
**Effect:** +1 to LOCK rolls (Lock-On weapons only)
**Use Case:** Makes establishing locks easier (58% → 72% success rate)

**Synergy:**
- Pair with Lock-On weapons
- Helps against Ghost Stealth System (-2 to lock)
- Useful for Javelin (3 DP heavy missile)

---

### 2. Reactive Armor Plating (eq_3)

**Type:** Reactive Defense System
**Effect:** Reduces 3 DP to 2 DP (blocks 1 DP from heavy weapons)
**Uses:** 2 per game

**Use Case:** Protects against heavy weapons (Javelin, Storm Shadow, Bunker Pile Driver)

**Synergy:**
- Pair with regular armor (+HP)
- Combine with shields (layered defense)
- Save uses for critical hits

---

### 3. Fortress Heavy Armor (eq_5)

**Type:** Defense System
**Effect:** +2 HP but -5cm movement
**Connector:** Can attach equipment to it

**Use Case:** Defensive builds that don't need mobility

**Trade-off:**
- Pure Legs: 15cm → 10cm
- Pure Wheels/Treads: 30cm → 25cm
- Gain +2 HP (50% more than base)

**Synergy:**
- Pair with Indirect Fire weapons (don't need to move)
- Combine with shields (maximum defense)
- Use with Lock-On weapons (stationary firing)

3. **Balanced** - Trade-offs and synergies
4. **Thematic** - Equipment fits its visual model
