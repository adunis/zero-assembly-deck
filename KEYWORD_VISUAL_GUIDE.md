## KEYWORD HIERARCHY

### Primary Keywords (Always First)
1. **Weapon Type** - Direct Fire, Indirect Fire, Lock-On, Melee
2. **Range** - Short, Medium, Long
3. **Damage** - 0 DP, 1 DP, 2 DP, 3 DP

### Secondary Keywords (After Primary)
4. **Combat Effects** - Multi-Shot, Heavy, Knockback, Piercing, Hazard, Reliable
5. **Defense** - Shield, Recharge, Reactive Armor, Heavy Armor
6. **Utility** - Free SCAVENGE, Damage +1, Fast Move, Jump Jets
7. **Countermeasure** - Countermeasure, Reactive, Active, Limited, Unlimited

### Example Order
```
**Direct Fire** • **Medium** (0-30cm) • **2 DP** • **Heavy** • **Piercing**
└─ Type ─────┘   └─ Range ──────────┘   └─ Dmg ┘   └─ Effects ──────────┘
```

## VISUAL STYLING

### Bold Keywords
- **Purpose:** Quick scanning during gameplay
- **Format:** `**Keyword**` in JSON → renders as bold in HTML
- **Color:** Black (default text color)
- **Weight:** 700 (bold)

### Italic Reminder Text
- **Purpose:** Brief explanation of mechanics
- **Format:** `<i>reminder text</i>` in JSON → renders as italic in HTML
- **Color:** #bbb (light gray)
- **Style:** Italic

### Bullet Separators
- **Purpose:** Visual rhythm between keywords
- **Character:** • (bullet point)
- **Spacing:** Space before and after bullet

### Line Breaks
- **Purpose:** Separate keywords from reminder text
- **Format:** `<br>` tag in JSON
- **Effect:** Creates visual separation

## PRINT OPTIMIZATION

When printed, cards maintain readability:

### Font Sizes (Print)
- **Card Title:** 7.5pt (bold)
- **Keywords:** 7pt (bold)
- **Reminder Text:** 7pt (italic)
- **Stat Line:** 6.5pt (bold)

### Colors (Print)
- **Keywords:** Black (high contrast)
- **Reminder Text:** Black (readable)
- **Borders:** Black (1.5px solid)

### Layout (Print)
- **Cards per page:** 9 (3×3 grid)
- **Card size:** 63mm × 88mm (poker size)
- **Spacing:** 6mm gap between cards
- **Paper:** A4 (210mm × 297mm)

## CONCLUSION

The keyword system transforms equipment cards from verbose paragraphs into scannable, modular components. Players can:

1. **Scan keywords** (bold) for instant understanding
2. **Read reminders** (italic) for clarification
3. **Build modular mechs** by combining keyword effects
4. **Learn once, apply everywhere** - consistent format

The visual hierarchy (bold keywords → italic reminders → stat line) guides the eye naturally from most important to least important information.
